    <h3>🚂Concept : 생성된 가상의 타겟 10개를 하트빔으로 빠르게 처리하는 게임</h3>
    <br>
    <h3>How To Play?</h3>
    <br>
    1. 앱이 실행되면 타겟 10개의 위치를 파악한다.
    <br>
    2. 손 하트 포즈를 카메라가 포커싱되게 하면 하트빔이 발사된다.(이때부터 시간 초가 재지기 시작)
    <br>
    3. 가운데 조준점에 맞춰서 하트빔을 발사시켜 타겟을 제거한다.
    <br>
    4. 10개를 다 맞춘다.
